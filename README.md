# Traffic Light Controller FSM

## Description
Designed and implemented a Traffic Light Controller using Verilog HDL and Finite State Machine (FSM) concepts. 
The controller manages traffic signal transitions based on predefined timing sequences and was verified 
using a testbench and GTKWave simulation.

## Tools Used
- Verilog HDL
- Icarus Verilog
- GTKWave
- Ubuntu Linux

## Files
- traffic_light.v
- tb_traffic_light.v
- traffic_waveform.png

## Concepts
- Finite State Machine (FSM)
- Sequential Logic Design
- State Transitions
- Traffic Signal Control
- Functional Verification

## Simulation
The design was simulated using Icarus Verilog, and the waveform results were analyzed using GTKWave to
verify correct traffic light state transitions.
